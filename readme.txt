A premium custom WordPress theme for Drama Mojaz, designed for sports news and drama content with high-performance aesthetics.

== Description ==

Drama Mojaz Theme is a modern, responsive WordPress theme specifically designed for high-traffic sports and news websites. Built with performance and elegance in mind, it features a glassmorphism design system.

Features:
* Responsive design that works on all devices (Mobile-First)
* RTL language support (Arabic optimized)
* Powerful Theme Options panel (Redux Framework)
* Instagram-style Stories System
* 11+ Premium Custom Widgets
* Modular Homepage Sections via Shortcodes
* Live Weather Integration (OpenWeatherMap)
* SEO optimized and Fast Loading

== Installation ==

1. Upload the theme files to your `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' menu in WordPress
3. Install and activate required plugins (Redux Framework)
4. Configure theme options in Theme Options panel

== Shortcodes ==

The theme provides shortcodes for all homepage sections:
* `[dm_homepage]` - All sections in order
* `[dm_hero]` - Hero Section
* `[dm_latest_articles]` - Latest Articles
* `[dm_stories]` - Stories Bar
* `[dm_videos]` - Video Section
* `[dm_advanced]` - Tabbed Category Section
* `[dm_compact]` - Grid View
* `[dm_trending]` - Trending Posts
* `[dm_zigzag]` - Alternating Layout
* `[dm_overlay]` - Special Reports
* `[dm_timeline]` - Chronological Timeline
